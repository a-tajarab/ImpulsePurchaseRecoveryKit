OcrFailureReason

enum OcrFailureReason : Enum<OcrFailureReason>

Describes the reason why OCR or receipt parsing failed.

Used to determine which error message, emoji, and action buttons are shown on the OcrFailureScreen. Each value represents a distinct stage of the scanning pipeline where failure can occur.

NO_TEXT_FOUND

ML Kit OCR ran successfully but returned no text from the image. Typically caused by a blurry photo, poor lighting, or an image that does not contain a receipt.

NO_ITEMS_PARSED

NO_ITEMS_PARSED

OCR successfully extracted text but the Claude AI parser could not identify any individual line items within it. Typically caused by an unusual or non-standard receipt format.

INCOMPLETE_RECEIPT

Items were parsed successfully but key fields such as the store name or total amount are missing from the result. The receipt is saved to the database with whatever data was found, and the user is directed to edit it from the Receipts screen.
