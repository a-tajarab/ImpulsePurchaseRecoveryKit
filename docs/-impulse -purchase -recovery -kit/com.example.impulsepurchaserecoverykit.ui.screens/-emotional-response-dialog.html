EmotionalResponseDialog

@Composable
fun EmotionalResponseDialog(response: EmotionalResponse, onDismiss: () -> Unit, onViewStats: () -> Unit)

Dialog composable that displays a compassionate emotional response when a user logs a high-regret purchase.

Shown by the EmotionalResponseEngine when a receipt is rated with a regret score at or above the high-regret threshold. The dialog is designed to be supportive rather than judgemental — it acknowledges how the user feels, provides a reassuring message, and offers a practical tip for avoiding a similar situation in the future.

The dialog contains three content areas:

  • A large emoji and headline that set the emotional tone at a glance

  • A body message that validates the user's feelings and offers encouragement

  • A tip card with one concrete, actionable suggestion for next time

Two actions are offered:

  • I'll do better — dismisses the dialog and returns to the previous screen

  • Show my stats — dismisses the dialog and navigates to the Stats screen so the user can review their spending and regret patterns in context

Parameters

response

The EmotionalResponse generated by EmotionalResponseEngine, containing the emoji, headline, message, and tip to display

onDismiss

Callback invoked when the user taps "I'll do better" or dismisses the dialog by tapping outside it

onViewStats

Callback invoked when the user taps "Show my stats" — typically navigates to the Stats screen
