reaction

val reaction: Int

The user's sentiment reaction to this item, stored as an integer. Possible values:

  • 1 = Happy — the user is glad they bought this item

  • 0 = Ok — the user feels neutral about this item

  • -1 = Regret — the user wishes they had not bought this item

The integer representation allows easy aggregation across all items on a receipt to produce the overall mood summary banner shown on the Receipt Detail screen.
