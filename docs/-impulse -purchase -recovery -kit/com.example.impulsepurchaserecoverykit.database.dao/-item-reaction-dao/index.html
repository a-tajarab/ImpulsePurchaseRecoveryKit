ItemReactionDao

interface ItemReactionDao

Data Access Object (DAO) for item reaction operations.

ItemReactionDao manages all database interactions for the per-item sentiment reactions that users assign on the Receipt Detail screen. Each reaction records whether the user felt Happy, Ok, or Regret about an individual item on a receipt.

Because users can update their reactions multiple times, most write operations use REPLACE or Upsert strategies to avoid duplicate entries. Flow-returning queries keep the UI in sync automatically as reactions are saved.
