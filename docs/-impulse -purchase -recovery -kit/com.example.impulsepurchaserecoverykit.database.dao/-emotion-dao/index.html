EmotionDao

interface EmotionDao

Data Access Object (DAO) for emotion check-in operations.

EmotionDao handles all database interactions related to the user's emotional responses to their purchases. Every time a user rates their regret or logs how they were feeling during a shopping trip, those entries are stored and retrieved through this DAO.

All Flow-returning queries are observed reactively — the UI automatically updates whenever the underlying emotion data changes in the database.
